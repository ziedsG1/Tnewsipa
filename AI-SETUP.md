# Article summaries (no API key)

Summaries are built **from the article page** in the **article’s own language** (Arabic, French, or English). No translation, no Groq, no Gemini, no API keys.

## How it works

1. Load the article HTML from the source link.
2. Detect language from the feed and article text.
3. Pick the most important sentences and show section labels in that same language.

## Build IPA

No AI secrets required. Push to `main` and run GitHub Actions as usual.
